# CONTROLE DE HORAS
## Sistema para gerenciar horas trabalhadas e extras usando o arquivo de texto do relógio da Dimep


### Descrição do problema
- A empresa possui o relógio de ponto da Dimep e o software de controle é muito antigo e não possui um controle eficaz de horas extras;
- O RH perde muito tempo tentando gerenciar e controlar as horas extras dos funcionários


### Tecnologias usadas
#### TALL Stack
- Laravel 12
- Livewire 3
- Tawilwind 4
- Alpine JS
